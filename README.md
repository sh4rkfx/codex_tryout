# TrainingPlan Android App

Dieses Repository enthält ein minimales Grundgerüst für eine native Android-App (Kotlin),
mit dem wir die Trainingsplan-Logik schrittweise aufbauen können.

## Inhalt
- Android-App-Modul (`app`)
- Basis-Theme, Farben, einfache `MainActivity`
- Beispiel-Layout für den Startbildschirm

## Nächste Schritte (Vorschlag)
1. Datenmodell für Nutzerprofil, Verletzungen, Prioritäten und Trainingshistorie.
2. Plan-Generator (Regelwerk) als separate Kotlin-Komponente.
3. Persistenz via Room + einfache UI für Profil-Setup.
